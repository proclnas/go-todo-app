Todo App - GO
---

A simple todo app to improve golang skills.

#### Usage

##### Install dependencies
```
$ dep ensure
```

##### Run main (dev, tests)
```
$ go run main.go
```

##### Compiled binary
```
$ go build
$ ./todo-app
```
